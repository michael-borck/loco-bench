https://github.com/huggingface/smollm  The Hugging Face SmolLM repository serves as a central hub for a collection of efficient and lightweight AI models designed to run directly on local devices. This family of tools includes both text-based language models and multimodal vision models that prioritise high performance within a compact digital footprint. A core theme of the project is its commitment to open-source transparency, providing not only the model weights but also the specific data mixtures and training configurations used. Ultimately, the resource aims to democratise powerful AI by offering multilingual support and advanced reasoning capabilities that do not require massive industrial hardware.

https://github.com/huggingface/smol-course  The smol-course is a practical, community-driven educational resource hosted on GitHub by Hugging Face that teaches users how to align and fine-tune small language models. By focusing on efficient models like SmolLM3, the curriculum allows developers to master advanced techniques—including instruction tuning, preference alignment, and synthetic data generation—using standard local hardware rather than expensive cloud infrastructure. The course is structured into distinct modules that guide learners from initial training to production deployment, emphasizing the benefits of small models such as enhanced privacy, lower operational costs, and reduced environmental impact. Ultimately, this repository serves as an accessible entry point for researchers and engineers to build customised, domain-specific AI through hands-on exercises and a collaborative, peer-reviewed framework.
